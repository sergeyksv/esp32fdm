ESP32 FDM Bridge

Turn a $15 ESP32-S3 board into a wireless 3D printer server with camera streaming and AI failure detection.

Features

  • MJPEG camera stream (800x600 @ 5 FPS)
  • USB Host serial bridge to any Marlin or Klipper printer
  • Obico cloud integration with AI failure detection
  • OctoPrint compatibility via RFC 2217
  • Web dashboard with temperature, progress, and terminal
  • SD card file management and host printing
  • WiFi AP fallback with captive portal for easy setup

Install Firmware

Connect your Freenove ESP32-S3-WROOM via the right USB-C port (UART) and click the button below.

Requirements: Chrome or Edge browser (Web Serial API). The left USB-C port is for printer connection, use the right one for flashing.

Tip: Hold the BOOT (download) button before clicking Install and keep holding it until the upload progress appears. Release it once flashing starts.

After Flashing

  • The board starts in AP mode — connect to the ESP32FDM-XXXX WiFi network
  • A captive portal opens automatically — enter your WiFi credentials
  • The board reboots and connects to your network — open its IP in a browser
  • Connect the left USB-C port to your 3D printer
